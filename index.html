⚡ One-time Firebase Setup Required

Your site needs a free Firebase database so data syncs across all devices. This takes 5 minutes and is completely free.

1
Go to Firebase Console
console.firebase.google.com → Create a new project (any name, e.g. "sciencewithabhi")
2
Add a Web App
In your project → click the web icon (</>) → Register app → Copy the firebaseConfig object
3
Enable Firestore
Left sidebar → Firestore Database → Create database → Start in test mode
4
Update this file
Open sciencewithabhi.html → Find firebaseConfig near the top → Replace the placeholder values with your real config → Save & redeploy

✓ Once configured, admin login will work with admin / admin123 and all posts will sync instantly to every device worldwide.
